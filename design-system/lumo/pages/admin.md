# Lumo admin page override

- Keep the master warm editorial palette and use the accessible action clay for primary controls.
- Desktop uses a 240px sidebar; widths below 840px use a dismissible off-canvas navigation and one-column content.
- Agent rows use system sans text, restrained borders, tabular status counts, and no decorative blur beyond navigation or drawer separation.
- Forms keep visible labels, inline errors, 46px minimum controls, a sticky save bar, Escape/scrim dismissal, and 3px keyboard focus rings.
- The identity prompt remains a monospaced long-form field; destructive actions are separated and confirmed.
- Motion is limited to 220ms state changes and disabled by `prefers-reduced-motion`.
